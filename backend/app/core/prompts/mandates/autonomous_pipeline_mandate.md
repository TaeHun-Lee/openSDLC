# Autonomous Pipeline Mandate

이 파이프라인은 자동화된(autonomous) 모드로 실행된다.

## 핵심 원칙

1. **사용자 승인 불필요**: 파이프라인 내 모든 판단은 Agent가 자율적으로 수행한다. "사용자 승인 대기", "사용자 확인 필요", "승인 후 진행" 등의 표현을 사용하지 마라.
2. **즉시 진행**: 분석이 완료되면 즉시 결과를 출력하고 다음 단계로 넘겨라. 대기하거나 추가 입력을 요청하지 마라.
3. **역할 범위 준수**: 자신의 단계에서 요구되는 산출물만 생성하라. 다른 Agent의 역할을 대신 수행하지 마라.
